# 更新日志

本项目采用[语义化版本](https://semver.org/lang/zh-CN/)记录版本。

## [1.1.0] - 2026-06-24

### 新增

- 支持原生文字层、扫描页和嵌入图片的离线打码。
- 支持跨行、跨页关键词匹配及命中统计。
- 增加可编辑遮盖预览、手动框选、颜色选择和另存为。
- 增加快速/全面 OCR 模式、加密 PDF 和输出安全校验。
- 提供 Windows 单文件 EXE 构建脚本与用户使用手册。
